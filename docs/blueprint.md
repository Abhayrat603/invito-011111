# **App Name**: E-motion Commerce

## Core Features:

- Intuitive Product Browsing: Allow users to easily browse products with filtering and sorting options.
- Seamless Add-to-Cart Experience: Enable users to add products to their cart quickly and efficiently.
- Secure Checkout Process: Implement a secure checkout process for processing orders via Firestore without external payment providers.
- Personalized User Account Dashboard: Provide a personalized dashboard where users can manage their orders, profile, and settings.
- Email/Password Authentication: Implement user authentication using email and password, with persistent sign-in via Firebase.
- Order Placement: Capture user orders in Firestore including user ID, cart items, and prices; create a thank you page upon completion.

## Style Guidelines:

- Primary color: Lively and modern hue drawn directly from globals.css (#222222).
- Background color: Very light gray (#f5f5f5), almost white, offering a clean backdrop.
- Accent color: Use a slightly darker gray/light-black tone, ensure adherence to existing color tokens.
- Headings Font: 'Geist', sans-serif, for headings.
- Body Font: 'Geist', sans-serif, for body text.
- Use clear, engaging icons that match the modern aesthetic. Icons should enhance usability and provide visual cues for different actions and categories.
- Employ a mobile-first, responsive layout using Flexbox and Grid as needed. Maintain consistency across devices with semantic tokens and spacing scales.
- Incorporate dynamic motion elements to enhance user experience. Use subtle animations for hover states, transitions, and loading sequences to maintain a professional feel.
